Cài đặt bảo vệ

Kích hoạt extension và kiểm soát các lớp bảo vệ.

Kích hoạt tài khoản

Lấy key tại Web App → Tài khoản → API & MCP key. Extension chỉ hoạt động sau khi key được xác minh.

Gateway

URL và nội dung được gửi đến địa chỉ này.

Tính năng

Quyền riêng tư

API key được lưu trong vùng lưu trữ riêng của extension và gửi qua Authorization header. Mỗi lượt quét được tính vào tài khoản tương ứng.
